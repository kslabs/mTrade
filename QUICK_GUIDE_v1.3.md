# 📖 БЫСТРОЕ РУКОВОДСТВО v1.3

## 🎯 Новый интерфейс

### 1. Header (Шапка):
```
┌────────────────────────────────────────────────────────────┐
│  Название      Режимы       Сервер + Управление           │
│                                                            │
│  🚀 mTrade     📊⚪🔄     🟢 PID:12345  00д 01:23:45  🔄⏹│
└────────────────────────────────────────────────────────────┘
     ↑              ↑                  ↑
     Слева       Центр              Справа
```

### 2. Автотрейдинг:
```
┌────────────────────────────────────────────────────────────┐
│  ⚪ ВЫКЛ    Котируемая: [USDT▼]    +123 USDT | 42 | 78%  │
└────────────────────────────────────────────────────────────┘
     ↑              ↑                      ↑
  Вкл/Выкл    Выбор валюты             Статистика
```

### 3. Закладки базовых валют:
```
┌────────────────────────────────────────────────────────────┐
│ [BTC₿] [ETH Ξ] [SOL ◎] [BNB🔶] [XRP✕] ... →              │
└────────────────────────────────────────────────────────────┘
```

---

## ⚙️ КАК ИСПОЛЬЗОВАТЬ

### Выбор торговой пары:

**Шаг 1:** Выберите котируемую валюту
```
Котируемая: [USDT ▼]
           └─ USDT, USDC, BTC, ETH
```

**Шаг 2:** Кликните на закладку базовой валюты
```
[BTC₿] ← Клик
```

**Результат:**
```
Торговая пара: BTC_USDT ✅
```

---

### Включение автотрейдинга:

**Шаг 1:** Выберите пару (см. выше)

**Шаг 2:** Кликните на переключатель
```
⚪ ВЫКЛ  →  🟢 ВКЛ
```

**Результат:**
```
АВТОТРЕЙДИНГ ВКЛ ✅
Статистика обновляется каждые 5 сек
```

---

### Переключение режимов:

**В центре header:**
```
📊 Обычный  ⚪→🟢  🔄 Копи
```

**Клик на переключатель:**
- Влево = Обычный трейдинг
- Вправо = Копитрейдинг

| BTC      | Bitcoin | ₿      |

| SOL      | Solana  | ◎      |
| BNB      | Binance | 🔶     |
| XRP      | Ripple  | ✕      |
| ADA      | Cardano | ₳      |
| AVAX     | Avalanche | 🔺   |
| DOT      | Polkadot | ⬤     |
| MATIC    | Polygon | 🔷     |

### Примеры пар:

```
BTC + USDT  =  BTC_USDT
ETH + USDC  =  ETH_USDC
SOL + BTC   =  SOL_BTC
BNB + ETH   =  BNB_ETH
```

## 💡 СОВЕТЫ

### 1. Быстрая смена пары:
```
1. Клик на закладку (например, ETH)
2. Автоматически: ETH_USDT
3. Баланс и ордера обновятся
```

### 2. Работа с разными котируемыми:
```
1. Котируемая: USDC
2. Клик на SOL
3. Результат: SOL_USDC
```

### 3. Мониторинг статистики:
```
Профит:   +123.45 USDT  (зеленый = прибыль)
         -45.67 USDT  (красный = убыток)
Сделки:   42           (всего выполнено)
Успешно:  78%          (процент успешных)
```

## 🎨 ВИЗУАЛЬНЫЕ ПОДСКАЗКИ

### Закладки:
```
[BTC]  ← Серая = неактивная
[ETH]  ← Зеленая = активная
[SOL]  ← Зеленая рамка при hover
```

### Автотрейдинг:
```
⚪ ВЫКЛ  ← Красная кнопка
🟢 ВКЛ   ← Зеленая кнопка
```

### Режимы:
```
📊 Обычный  ← Активен (зеленый текст)
🔄 Копи     ← Неактивен (серый текст)
```

## 🔄 ЧТО ПРОИСХОДИТ ПРИ СМЕНЕ ПАРЫ

### Автоматически:
1. ✅ Обновляется поле "Торговая пара"
2. ✅ Перезагружается баланс
3. ✅ Перезагружаются ордера
4. ✅ Показывается уведомление
5. ✅ Подсвечивается активная закладка

### Пример:
```
[Клик на ETH]

→ Пара: ETH_USDT
→ Баланс обновлен
→ Ордера обновлены
→ ✅ "Выбрана пара ETH_USDT"
```

## ⚡ ГОРЯЧИЕ КЛАВИШИ

_В разработке_

Планируется:
- `1-9, 0` - Быстрое переключение закладок
- `Space` - Вкл/Выкл автотрейдинга
- `M` - Смена режима
- `Q` - Смена котируемой валюты
