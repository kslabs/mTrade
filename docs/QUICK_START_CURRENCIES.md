# ⚡ Быстрый старт: Управление валютами

## 🎯 Как настроить свой список валют

### Шаг 1: Откройте менеджер валют
```
┌────────────────────────────────────────────────────┐
│ [BTC₿] [ETH Ξ] [SOL ◎] ... [⚙️] ← НАЖМИТЕ СЮДА   │
└────────────────────────────────────────────────────┘
```

### Шаг 2: Редактируйте существующие
```
[BTC      ] [₿        ] [🗑️]  ← Изменить код/символ
[ETH      ] [Ξ        ] [🗑️]
[SOL      ] [◎        ] [🗑️]
```

### Шаг 3: Добавьте новые
```
[➕ Добавить валюту] ← Нажмите
```

### Шаг 4: Сохраните изменения
```
[Отмена] [Сохранить] ← Нажмите "Сохранить"
```

## ⚠️ Важно знать

✅ **Можно:**
- Добавлять сколько угодно валют
- Редактировать коды и символы
- Удалять ненужные валюты
- Использовать emoji как символы

❌ **Нельзя:**
- Удалить последнюю валюту
- Создать дубликаты кодов
- Оставить код пустым

## 🔄 Быстрые действия

### Добавить Dogecoin:
1. ⚙️ → ➕ Добавить валюту
2. Код: `DOGE`, Символ: `🐕`
3. Сохранить

### Удалить валюту:
1. ⚙️ → Найти валюту
2. 🗑️ Удалить → Подтвердить
3. Сохранить

### Сбросить к дефолту:
1. F12 (консоль браузера)
2. `localStorage.removeItem('baseCurrencies')`
3. F5 (обновить страницу)

## 📱 Горячие клавиши

- **ESC** - Закрыть модальное окно
- **Клик вне окна** - Закрыть без сохранения
- **Enter в поле** - Переход к следующему полю
