# ✅ ГОТОВО: СИНЕЕ НАЗВАНИЕ ДЛЯ НЕАКТИВНЫХ ВАЛЮТ

## 🎯 ЧТО СДЕЛАНО

Реализована новая визуальная индикация неактивных валют:
- **Вместо синего бордюра** → **Ярко-синий цвет названия валюты**

---

## 🔑 КЛЮЧЕВОЙ МОМЕНТ

```javascript
// Валюта НЕАКТИВНА, если:
permissionEnabled !== true

// Результат:
.code-label { color: #2196F3 !important }  // ← СИНИЙ текст
```

---

## 📝 КРАТКАЯ ЛОГИКА

```
ЕСЛИ торговля НЕ разрешена явно:
  → Название валюты = СИНИЙ цвет 🔵
  → Бордюр = стандартный (зелёный)
ИНАЧЕ (торговля разрешена):
  → Название валюты = белый/серый ⚪
  → Бордюр = по цене (🟡 жёлтый / 🔴 красный / 🟢 зелёный)
```

---

## 🎨 ВИЗУАЛЬНО

### Неактивная валюта:
```
┌─────────────────┐
│ 🔴 ANIME 🌸     │  ← СИНЕЕ название
└─────────────────┘     Обычный бордюр
```

### Активная валюта:
```
┏━━━━━━━━━━━━━━━━━┓
┃ 🟢 BTC ₿        ┃  ← БЕЛОЕ название
┗━━━━━━━━━━━━━━━━━┛     Цветной бордюр
```
