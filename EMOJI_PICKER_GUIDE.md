# 😀 Руководство по выбору пиктограмм валют

## Обзор функции

В редакторе валют теперь доступен удобный **Emoji Picker** для выбора символов/пиктограмм для каждой валюты. Это позволяет визуально идентифицировать валюты на интерфейсе.

---

## Как использовать

### 1. Открыть редактор валют
- Нажмите кнопку **⚙️** (шестерёнка) справа от вкладок валют

### 2. Выбрать пиктограмму
- Для **новой валюты**: нажмите кнопку **😀** рядом с полем "Символ"
- Для **существующей валюты**: кликните на поле "Символ" или на кнопку **😀**

### 3. В открывшемся Emoji Picker
Вы можете:
- **Выбрать из популярных символов** - кликните на любой символ в сетке
- **Ввести свой символ** - введите в поле внизу и нажмите **✓**

### 4. Сохранить изменения
- Нажмите **💾 Сохранить** для применения изменений

## Популярные символы для криптовалют

### Официальные символы
- **₿** - Bitcoin (BTC)
- **Ξ** - Ethereum (ETH)
- **◎** - Solana (SOL)
- **₳** - Cardano (ADA)
- **✕** - Ripple (XRP)

### Геометрические фигуры
- **🔶** - Binance Coin (BNB)
- **🔺** - Avalanche (AVAX)
- **⬤** - Polkadot (DOT)
- **💠** - Dogecoin (DOGE)
- **🔷** - Tron (TRX)
- **🔰** - Gate Token (GT)

### Тематические эмодзи
- **💎** - Драгоценность (для премиум-токенов)
- **🚀** - Растущие монеты
- **⚡** - Быстрые транзакции (Lightning, Solana)
- **🔥** - Популярные/горячие токены
- **🌐** - Глобальные/интернациональные проекты (Worldcoin)
- **🪙** - Универсальный символ монеты

### Финансовые символы
- **💰** - Деньги/богатство
- **💵** - Долларовые стейблкоины
- **💴** - Азиатские токены
- **💶** - Европейские токены
- **💷** - Британские токены

## Рекомендации

### ✅ Хорошие практики
- Используйте **узнаваемые символы** для популярных монет
- Для новых токенов выбирайте **тематические эмодзи**
- Избегайте **сложных многосимвольных эмодзи** (могут плохо отображаться)
- Проверяйте отображение на разных устройствах

### ❌ Чего избегать
- Слишком похожих символов для разных валют
- Символов, которые плохо видны на тёмном фоне
- Длинных текстовых строк (макс. 4 символа)
- Специальных управляющих символов

## Технические детали

### Структура данных
```json
{
  "code": "BTC",
  "symbol": "₿"
}
```

### Поддерживаемые символы
- **Unicode-символы** (₿, Ξ, ◎, ✕, ₳)
- **Emoji** (🔶, 💎, 🚀, ⚡)
- **ASCII-символы** (латиница, цифры)
- **Максимальная длина**: 4 символа

### Хранение
- Символы сохраняются в `currencies.json`
- Кодировка: UTF-8
- Автоматическая синхронизация с UI
