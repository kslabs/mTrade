# ❗ ВАЖНО: Где смотреть логи

## Проблема
Вы смотрите логи **браузера** (консоль разработчика F12), а мне нужны логи **сервера Python**!

## Где найти логи сервера

### Вариант 1: Окно PowerShell/CMD
Когда вы запускаете `RESTART.bat` или `mTrade.py`, открывается окно PowerShell/CMD.

**ВОТ ЭТО ОКНО МНЕ НУЖНО!**

В нём должны быть такие логи:
```
[AutoTraderV2] Главный цикл запущен, quote=USDT
[ETH] Начало обработки...
[ETH] Цена: 3038.01
[ETH] Цикл НЕ АКТИВЕН → попытка стартовой покупки
[ETH] [DEBUG] Получаем параметры торговли...
[ETH] [DEBUG] Параметры получены: start_volume=10.0
...
```

### Вариант 2: Если окна нет
Если вы закрыли окно или логи не видны:

1. **Откройте новое PowerShell**
2. **Перейдите в папку проекта:**
   ```powershell
   cd "C:\Users\Администратор\Documents\bGate.mTrade"
   ```
3. **Посмотрите текущие процессы:**
   ```powershell
   python check_processes.py
   ```

## Как отправить мне логи

### Способ 1: Скриншот
Сделайте скриншот **окна PowerShell**, где работает сервер

### Способ 2: Скопировать текст
1. Щёлкните правой кнопкой на окне PowerShell
2. Выберите "Пометить" (Mark)
3. Выделите текст
4. Нажмите Enter (скопируется)
5. Вставьте сюда

### Способ 3: Перенаправить в файл
Запустите сервер с перенаправлением:
```powershell
python mTrade.py > server_output.txt 2>&1
```

Затем откройте `server_output.txt` и скопируйте мне.

## Что конкретно нужно

Логи от момента:
```
[ETH] Начало обработки...
```

До:
```
[ETH] [OK] Покупка записана в лог
```

Или до места, где логи обрываются.
