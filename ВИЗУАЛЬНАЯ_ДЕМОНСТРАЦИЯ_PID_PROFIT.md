## 📐 Размеры и отступы

### ДО:
```
┌─────────────────┐
│ ⬆️ 6px         │
│                 │
│  🟢 PID: 3012  │  ⬅️ Одна строка
│                 │
│ ⬇️ 6px         │
└─────────────────┘
```

### ПОСЛЕ:
```
┌─────────────────┐
│ ⬆️ 4px         │  ⬅️ Компактнее
│                 │
│  🟢 PID: 3012  │  ⬅️ Строка 1
│      Profit:    │  ⬅️ Строка 2
│      +12.34     │
│                 │
│ ⬇️ 4px         │  ⬅️ Компактнее
└─────────────────┘
```

**Экономия места:** `(6-4)×2 = 4px` по вертикали

## 🔄 Динамика обновления

```
Время 00:00:00  →  Profit: 0.00    (Серый)
Время 00:05:00  →  Profit: +2.15   (Зелёный)
Время 00:10:00  →  Profit: +8.34   (Зелёный)
Время 00:15:00  →  Profit: +12.34  (Зелёный)
Время 00:20:00  →  Profit: -3.21   (Красный)
        ⬆️
Обновление каждые 5 секунд
```
