===============================================================================
  🎯 КЛЮЧЕВЫЕ ОСОБЕННОСТИ ПРОЕКТА
===============================================================================

✨ Два режима торговли: Обычный и Копитрейдинг
⚡ Автотрейдинг с настраиваемыми параметрами
💱 Управление валютными парами с редактированием
📊 WebSocket real-time данные (orderbook, trades, ticker)
💾 Серверное сохранение всех настроек и состояния UI
🔄 Автоматическое восстановление после перезагрузки
🌐 Полная русская локализация (код, комментарии, документация)

===============================================================================
  🛠️ ТЕХНОЛОГИИ
===============================================================================

Backend:  Python 3.8+ | Flask 2.3.3 | WebSocket | RESTful API
Frontend: HTML5 | CSS3 (BEM) | JavaScript ES6+ | Fetch API | WebSocket API
DevOps:   Git | GitHub Actions (CI/CD) | pytest

===============================================================================
  📊 УПРАВЛЕНИЕ СЕРВЕРОМ
===============================================================================

Windows BAT файлы:
   START.bat              - Запустить сервер
   STOP.bat               - Остановить сервер
   RESTART.bat            - Перезапустить сервер
   STATUS.bat             - Проверить статус

Python скрипты (кроссплатформенные):
   python start.py        - Запустить
   python stop.py         - Остановить
   python restart.py      - Перезапустить
   python status.py       - Статус

Веб-интерфейс:
   🔄 Кнопка перезапуска в header (справа)
   ⏹️ Кнопка остановки в header (справа)
