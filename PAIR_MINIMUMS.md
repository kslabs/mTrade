# Справочник минимальных объёмов ордеров (PAIR_MINIMUMS)

Этот файл описывает минимальные объёмы (min_base_amount) для базовых валют, используемые приложением mTrade как статический/репозиторный источник, когда биржа не возвращает значения `min_base_amount` / `min_quote_amount`.

Формат: JSON файл `pair_minimums.json` в корне репозитория. Пример записи:

{
  "ETH": {"min_base_amount": "0.001", "note": "Минимальный объём базовой валюты для ETH"}
}

Правила и заметки:
- Значения `min_base_amount` указаны в единицах базовой валюты (например ETH, BTC и т.д.).
- `min_quote_amount` вычисляется динамически при необходимости как `min_base_amount * price` (цена берётся из рыночных данных / тикера).
- Этот репозиторный источник служит *fallback* для случая, когда API биржи не возвращает минимальные объёмы.
- Комментарии и пояснения находятся в этом файле (`PAIR_MINIMUMS.md`), сам `pair_minimums.json` хранит только данные в формате JSON (без комментариев).

Если нужно добавить/обновить минимумы для новой валюты — добавьте запись в `pair_minimums.json` и краткое пояснение в этом файле на русском языке.

---

Файл интегрирован в `mTrade.py` — серверный код использует значения из `pair_minimums.json` перед попыткой вычислить min-значения из orderbook/ticker.
