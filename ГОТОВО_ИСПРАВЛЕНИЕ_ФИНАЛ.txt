## 🎯 ВЫПОЛНЕНО

### 1. Исправлена раскраска в синий цвет ✅

**Решение:** Усилены CSS правила с `!important` и дополнительными селекторами

### 2. Прижато название к левому краю ✅
**Проблема:** Название было по центру  
**Решение:** Изменён `justify-content: flex-start` + отступ 2px

## 🔧 ИЗМЕНЕНИЯ

### static/style.css (2 места)

#### 1. Выравнивание и отступ:
```css
.base-currency-tabs .tab-item {
  padding: 4px 18px 4px 2px;        /* было: 4px 18px 4px 10px */
  justify-content: flex-start;       /* было: center */
}
.base-currency-tabs .tab-item .code-label {
  margin-left: 2px;                  /* НОВОЕ: 2px от края */
}
```

#### 2. Усиленные правила синего цвета:
```css
/* Синий с эффектом свечения */
.base-currency-tabs .tab-item.inactive-currency .code-label {
  color: #2196F3!important;
  text-shadow: 0 0 2px rgba(33,150,243,0.5)!important;
}

/* Дополнительные селекторы для гарантии */
.tab-item.inactive-currency .code-label,
.base-currency-tabs .inactive-currency .code-label {
  color: #2196F3!important;
}
```

### static/app.js (1 место)

#### Детальное логирование:
```javascript
const isInactive = (permissionEnabled !== true);

console.log(`[CURRENCY_COLOR] ${code}: isInactive=${isInactive}, ...`);

if(isInactive){
  tab.classList.add('inactive-currency');
  console.log(`[CURRENCY_STATUS] ${code}: INACTIVE - добавлен класс inactive-currency, название ДОЛЖНО быть синим!`);
  
  const hasClass = tab.classList.contains('inactive-currency');
  const codeLabel = tab.querySelector('.code-label');
  console.log(`[DEBUG] ${code}: classList contains inactive-currency = ${hasClass}, .code-label найден = ${!!codeLabel}`);
}
```
