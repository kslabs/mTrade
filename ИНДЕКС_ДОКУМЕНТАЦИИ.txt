    📊 КЛЮЧЕВЫЕ КОНЦЕПЦИИ
═══════════════════════════════════════════════════════════════

🔹 MARKET-ордер (покупка):
   • Исполняется СРАЗУ по рыночной цене
   • Гарантированное исполнение
   • Может быть slippage (отклонение от расчётной цены)
   • Таблица пересчитывается с РЕАЛЬНОЙ ценой после покупки

🔹 LIMIT-ордер (продажа):
   • Исполняется ТОЛЬКО по цене >= указанной
   • Гарантия минимальной цены
   • Может НЕ исполниться сразу (ждёт в книге заявок)
   • Исключает slippage вниз

🔹 target_sell_price (целевая цена продажи):
   • Рассчитывается от РЕАЛЬНОЙ цены покупки (start_price)
   • Формула: start_price × (1 + target_delta_pct / 100)
   • Гарантирует минимальную прибыль согласно таблице
   • НЕ изменяется после расчёта (защита от подмены)

🔹 breakeven_price (цена безубыточности):
   • Рассчитывается с учётом всех комиссий
   • Минимальная цена для покрытия затрат
   • target_sell_price ВСЕГДА должна быть >= breakeven_price

🔹 Цикл торговли:
   • Стартовая покупка → Ожидание → Продажа → Новый цикл
   • Каждый цикл независим
   • Инкрементальный ID для отслеживания
