## ⚡ ПРОВЕРКА (2 МИНУТЫ)

### 1. Перезапустить бота
```powershell
python _autotrader.py
```

### 2. Открыть и ЖЁСТКО обновить браузер
```
http://localhost:8080
Ctrl + Shift + R  (ОБЯЗАТЕЛЬНО!)
```

### 3. Проверить визуально
**ANIME (если неактивна):**
- ✅ Название **ЯРКО-СИНЕЕ**
- ✅ Прижато **К ЛЕВОМУ КРАЮ** (2px от бордюра)

---

## 🔍 ЕСЛИ НЕ РАБОТАЕТ

### Проверка 1: Консоль (F12)
Должны быть логи:
```
[CURRENCY_STATUS] ANIME: INACTIVE - добавлен класс inactive-currency
```

### Проверка 2: Element Inspector (F12 → Elements)
```
1. Найдите: <div class="tab-item" data-code="ANIME">
2. Класс "inactive-currency" должен быть в списке
3. Внутри: <span class="code-label">ANIME</span>
4. Computed стили: color = rgb(33, 150, 243)
```

### Проверка 3: Кэш
```
Ctrl + Shift + R (жёсткое обновление)
Или откройте в режиме инкогнито (Ctrl + Shift + N)
```
